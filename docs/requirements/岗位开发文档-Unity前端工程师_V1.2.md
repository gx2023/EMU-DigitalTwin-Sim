## 1. 岗位职责概述

### 1.1 核心职责

Unity前端开发工程师是本项目的核心开发角色之一，主要负责以下工作：

| 职责领域 | 具体内容 |
|---------|---------|
| **3D场景搭建** | 基于Unity引擎进行段厂3D场景的创建、编辑、管理和展示，包括场景LOD管理、分层管理、漫游功能等 |
| **模型导入优化** | 负责外部3D模型（FBX/OBJ/glTF等格式）的导入、材质配置、LOD设置、性能优化等工作 |
| **交互开发** | 开发3D场景中的交互功能，包括对象选择、信息展示、测量标注、截图录屏、场景搜索定位等 |
| **渲染优化** | 负责URP渲染管线配置、光照烘焙、后处理效果、粒子特效、阴影优化等，确保满足帧率要求 |
| **检修仿真开发** | 开发动车组调车仿真、部件检修仿真（转向架/制动/牵引等）、作业过程动画仿真等功能 |
| **多终端适配** | 负责PC端、大屏端（4K/8K）、VR端的多终端适配开发和布局适配 |
| **VR/AR开发** | 开发VR检修培训场景、AR检修辅助功能，适配主流VR/AR设备 |
| **数据对接** | 与后端工程师协作完成数据接口对接，实现实时数据在3D场景中的展示和驱动 |
| **UI框架搭建** | 搭建Unity UI框架，开发数据看板、信息面板等2D/3D混合界面 |

### 1.2 协作关系

| 协作对象 | 协作内容 |
|---------|---------|
| 后端工程师 | API接口对接、实时数据推送、数据格式约定 |
| 3D美术师 | 模型导入规范、材质配置、动画绑定、LOD设置 |
| 算法工程师 | 路径规划算法集成、调度算法结果可视化 |
| AI工程师 | AI识别结果在3D场景中的展示和标注 |
| 产品经理 | 需求评审、交互方案确认、验收标准对齐 |
| 测试工程师 | 性能测试、兼容性测试、VR体验测试 |
| 业务分析师 | 检修工艺流程确认、仿真逻辑验证 |

### 1.3 技能要求

| 技能类别 | 具体要求 |
|---------|---------|
| Unity引擎 | 熟练掌握Unity 2022 LTS或Unity 6，了解URP渲染管线 |
| 编程语言 | 精通C#，熟悉Unity脚本生命周期和设计模式 |
| 3D图形学 | 了解渲染管线、着色器（Shader）、光照模型、PBR材质 |
| 性能优化 | 熟悉Draw Call优化、LOD策略、遮挡剔除、GPU Instancing等 |
| XR开发 | 了解OpenXR、VR/AR交互设计、空间定位 |
| 资源管理 | 熟悉Addressables资源管理、AssetBundle |
| UI开发 | 熟悉Unity UI Toolkit / UGUI，了解响应式布局 |
| 网络通信 | 了解WebSocket、RESTful API、MQTT等通信协议 |

## 2. 段厂仿真模块需求（完整）

### 2.1 FR-001 段厂3D场景管理

**功能描述：**

段厂3D场景管理功能提供动车组运用所/动车段的三维场景创建、编辑、管理和展示能力，是数字孪生系统的基础空间框架。

**功能需求明细：**

| 需求编号 | 需求名称 | 需求描述 | 优先级 |
|---------|---------|---------|--------|
| FR-001-01 | 段厂3D场景创建 | 支持基于CAD图纸/点云数据/实景照片创建段厂3D场景模型 | P0 |
| FR-001-02 | 场景LOD管理 | 支持多级LOD（Level of Detail）模型管理，根据视距自动切换模型精度 | P0 |
| FR-001-03 | 场景分层管理 | 支持按建筑、轨道、设备、车辆、管线等层级管理场景元素 | P0 |
| FR-001-04 | 场景编辑工具 | 提供场景编辑工具，支持模型导入、位置调整、属性编辑等操作 | P1 |
| FR-001-05 | 场景漫游 | 支持第一人称/第三人称视角的场景漫游，支持路径规划和自动漫游 | P0 |
| FR-001-06 | 场景标注 | 支持在3D场景中添加文字标注、热点、测量标注等 | P1 |
| FR-001-07 | 场景搜索定位 | 支持按名称/编号快速搜索并定位到场景中的对象 | P0 |
| FR-001-08 | 场景截图与录屏 | 支持场景截图和视频录制，用于报告和培训 | P1 |
| FR-001-09 | 昼夜模式切换 | 支持白天/夜间场景模式切换，模拟不同光照条件 | P2 |
| FR-001-10 | 天气效果模拟 | 支持晴天、雨天、雪天等天气效果模拟 | P2 |
| FR-001-11 | 场景版本管理 | 支持场景模型的版本管理和变更记录 | P1 |
| FR-001-12 | 场景导入导出 | 支持场景模型的导入导出（FBX/OBJ/glTF等格式） | P0 |

**数据要求：**

- 段厂3D模型数据（建筑、轨道、设备等）
- 场景配置数据（光照、天气、标注等）
- 场景元数据（创建时间、修改时间、版本号等）

**交互要求：**

- 场景加载时间不超过10秒（LOD1级别）
- 场景漫游帧率不低于30FPS
- 支持鼠标、键盘、游戏手柄等多种输入方式

**前端开发要点：**

- 实现场景LOD自动切换系统（LOD0: 0-50m, LOD1: 50-200m, LOD2: 200-500m, LOD3: >500m）
- 开发场景分层管理系统，支持按类型过滤和显隐控制
- 实现第一人称/第三人称相机控制器，支持平滑过渡
- 开发路径规划漫游功能，支持关键帧动画
- 实现场景标注系统（文字标注、热点标注、测量标注）
- 开发搜索定位功能，相机平滑飞行至目标对象
- 实现场景截图（高分辨率）和视频录制功能
- 开发昼夜模式切换（Lighting Settings切换 + 天空盒切换）
- 实现天气粒子效果系统（雨、雪等）

### 2.2 FR-002 动车组调车仿真

**功能描述：**

动车组调车仿真功能模拟动车组在段厂内的调车作业过程，包括入库、转线、出库等操作，为段厂调度管理提供仿真验证工具。

**功能需求明细：**

| 需求编号 | 需求名称 | 需求描述 | 优先级 |
|---------|---------|---------|--------|
| FR-002-01 | 调车路径规划 | 支持基于段厂轨道拓扑的自动调车路径规划 | P0 |
| FR-002-02 | 调车过程仿真 | 支持动车组沿规划路径的调车过程动画仿真 | P0 |
| FR-002-03 | 调车冲突检测 | 支持多车同时调车时的路径冲突检测和预警 | P0 |
| FR-002-04 | 调车速度模拟 | 支持模拟不同调车速度（5km/h、10km/h、15km/h等） | P1 |
| FR-002-05 | 调车时间估算 | 支持基于路径和速度的调车时间自动估算 | P0 |
| FR-002-06 | 手动调车操作 | 支持用户手动控制动车组在轨道上的移动 | P1 |
| FR-002-07 | 调车计划导入 | 支持从外部系统导入调车计划数据 | P1 |
| FR-002-08 | 调车历史回放 | 支持历史调车过程的三维回放 | P2 |
| FR-002-09 | 信号系统仿真 | 支持段厂信号系统的状态仿真（进路、信号机等） | P1 |
| FR-002-10 | 防溜措施仿真 | 支持防溜措施的施加和撤除仿真 | P1 |

**业务规则：**

- 调车速度在段厂内不得超过15km/h
- 同一股道同时只允许一列动车组占用
- 道岔转换必须在动车组到达前完成
- 调车作业必须经过信号系统授权

**前端开发要点：**

- 开发轨道拓扑数据结构，实现轨道段和道岔的3D可视化
- 实现动车组沿轨道运动的动画系统（路径跟随、道岔切换）
- 对接后端A*路径规划算法，在3D场景中可视化展示规划路径
- 开发多车调车冲突检测的3D可视化预警（碰撞预警标注、颜色高亮）
- 实现调车速度控制UI和动画速度调节
- 开发手动调车模式（键盘/手柄控制动车组移动）
- 实现信号系统3D模型状态联动（信号机颜色变化、进路标识）
- 开发调车历史回放功能（时间轴控制、进度条拖拽）

| LOD多级细节 | 根据视距自动切换模型精度（LOD0-LOD3） | 段厂全景、设备模型 |
| 遮挡剔除 | 剔除被遮挡物体，减少Draw Call | 室内场景、密集设备区域 |
| 实例化渲染 | GPU Instancing渲染相同模型 | 股道、标准化设备 |
| 光照烘焙 | 预计算静态光照，减少实时计算 | 段厂建筑、固定设施 |
| 纹理压缩 | 使用ASTC/BC7压缩纹理格式 | 所有场景 |
| 异步加载 | Addressables异步加载资源 | 场景切换、模型加载 |
| 合批处理 | Static Batching / Dynamic Batching | 静态场景元素 |
| 相机裁剪 | 设置合理的相机裁剪面 | 所有场景 |

### 8.2 VR场景性能指标

### 9.3 LOD策略

| LOD级别 | 视距范围 | 面数比例 | 用途 |
|---------|---------|---------|------|

| LOD1 | 50-200m | 50% | 中景浏览 |
| LOD2 | 200-500m | 25% | 远景概览 |
| LOD3 | >500m | 10% | 全景视图 |

**LOD切换规则：**
