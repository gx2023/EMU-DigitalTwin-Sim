# 动车组检修数字孪生仿真系统 - Unity前端开发计划

本项目旨在构建基于Unity引擎的动车组检修数字孪生仿真系统，结合Web前端技术，为用户提供直观、高效的仿真界面体验。

## 技术架构说明

### Web前端技术栈
- **框架**: HTML5 + Tailwind CSS + Chart.js + Three.js
- **设计风格**: 工业科技风、深蓝色主题、网格背景
- **功能**: 已实现功能：
  - 实时数据看板
  - 3D可视化（Three.js）
  - 数据图表
  - 作业进度管理
  - 设备状态监控
  - 预警信息展示
  - 粒子特效
  - 粒子背景

### Unity前端技术栈
- **引擎**: Unity 2022.3 LTS
- **语言**: C#
- **渲染**: URP/HDRP
- **VR支持**: Oculus/Meta Quest, HTC Vive
- **AR支持**: AR Foundation
- **物理引擎**: PhysX
- **网络**: Unity Netcode for GameObjects
- **UI系统**: UIE
- **状态管理**: Scriptable Objects
- **资源管理**: Addressables

## Web前端与Web端集成方案

### 集成策略
1. **Web端展示与Unity通过WebSocket实时数据通信
2. **Unity提供3D渲染，Web负责数据展示
